# 🧬 Life RPG: The Simulation (人生模擬器)

這是一個基於 Streamlit 開發的回合制人生模擬遊戲。玩家需要管理時間、金錢與能量，在工作、學習與健康之間取得平衡。

## 🎮 遊戲特色

- **四大屬性**：❤️ 健康、🧠 智力、💰 財富、😊 心情 (Sanity)。
- **回合制玩法**：每個月分配 100 點能量進行活動。
- **隨機事件**：這就是人生！可能會突然生病、車子壞掉，或是中樂透。
- **復古風格**：使用 Streamlit 原生元件打造的 Pixel/Retro 風格介面。

## 🚀 如何執行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🛠 技術堆疊

- Python
- Streamlit
- Pandas
